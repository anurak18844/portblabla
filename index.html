    Portfolio นี้เขียนขึ้นเพื่อใช้สำหรับการสมัครเรียน การสมัครงาน ตลอดจนรวบรวมผลงาน และประสบการณ์การทำงานต่าง ๆ ของตัวข้าพเจ้า ผู้อ่านจะสามารถรับรู้สิ่งที่ได้กล่าวมาข้างต้นจาก Portfolio นี้

แนะนำตัวเอง

    สวัสดีครับ ผม นายอนุรักษ์ ศิวะบุญญา ชื่อเล่น บาส อายุ 20 ปีบริบูรณ์ ตอนนี้กำลังศึกษาในระดับประกาศนียบัตรวิชาชีพชั้นสูง(ปวส.) ชั้นปีที่ 2 สาขาวิชาเทคโนโลยีสารสนเทศ มีความรู้ในด้านการพัฒนาเว็ปไซต์ด้วยภาษา HTML CSS BOOTSTARP4, 5 JAVA ร่วมกับฐานข้อมูลอย่าง MySQL การเขียนโปรแกรมด้วยภาษา Python C C++ และการเขียนโปรแกรมเชิงวัตถุด้วยภาษา JAVA และตอนนี้อยู่ในช่วงของการฝึกงานในด้านการพัฒนาแอปพลิเคชั่นบนมือถือด้วย Flutter + Dart ซึ่งทักษะต่างๆ อยู่ในระดับที่สามารถต่อยอดได้ โดยส่วนตัวเป็นคนที่รู้หน้าที่ ทั้งการใช้ชีวิต การทำงาน และการเรียนรู้ มุ่งเน้นเป้าหมาย และการพัฒนาตนเอง และสามารถจัดการกับเวลาได้เป็นอย่างดี

บุคลิกภาพ

  • มีความเป็นผู้นำ
    • สามารถจัดการกับอารมณ์และผู้คนของตนเองได้อย่างมีประสิทธิภาพ และพร้อมที่จะให้คำแนะนำอยู่เสมอ
  • มีมนุษย์สัมพันธ์ดี
    • เป็นคนที่ชอบรับฟัง สามารถพูดคุยและเข้ากับผู้อื่นได้เป็นอย่างดี ทั้งการทำงานและการเรียนรู้ร่วมกันกับผู้อื่น และให้เกียรติผู้อื่นอย่างจริงใจ
  • เตรียมพร้อมในการเรียนรู้อยู่เสมอ
    • จะรู้สึกดีใจ ตื่นเต้น เมื่อได้เรียนรู้สิ่งใหม่ๆ มีความใฝ่รู้ในตัวเอง สามารถจัดการกับเวลาได้เป็นอย่างดี มีความรับผิดชอบในหน้าที่

โปรเจคที่เคยทำ

  • เว็บนิทรรศการออนไลน์
    • พัฒนาด้วยภาษา JAVA + MySQL ร่วมกับ HTML + CSS และ BOOTSTRAP4 และใช้การแชร์ไลฟ์สดแบบฝังในการถ่ายทอดสด
  • เว็บจองตั๋วหนัง
    • พัฒนาด้วยภาษา PHP + MySQL สามารถ เพิ่ม ลบ แก้ไข ข้อมูล User และข้อมูลหนัง อาทิ รอบฉาย ชื่อ รูปภาพ ผู้ใช้งานทั่วไปสามารถ ลงทะเบียน ล็อคอิน แก้ไขข้อมูลส่วนตัว ดูรอบฉาย และจองตั๋วหนังล่วงหน้าผ่านเว็ปไซต์ และชำระที่หน้าเคาน์เตอร์ ผู้ดูแลสามารถรับชำระผ่านหน้า POS ของเว็บไซต์ ตรวจสอบสถาณะ หรือประวัติการจอง และการชำระค่าตั๋ว
  • Durian Meter
    • แอปพลิเคชั่นวัดค่าความสุกของทุเรียนโดยใช้เสียงเคาะ ทำในระหว่างฝึกวิชาชีพ รับผิดชอบในส่วนของ UI ทำการรับ-ส่งข้อมูล และนำมาแสดงโดยใช้ RESTAPI พัฒนาด้วย Flutter + Dart
  • โมเดลวิเคราะห์ความคิดเห็นของนักศึกษาบนทวิตเตอร์โดยใช้การประมวลผลภาษาธรรมชาติ และการเรียนรู้ของเครื่องจักร
    • โมเดลที่วิเคราะห์การจำแนกประเภทข้อความที่ดึงมาจากทวิตเตอร์ โดยจะแนกเป็น 4 ประเภท ได้แก่ ได้แก่ การเรียนการสอน สิ่งอํานวยความสะดวก ทรัพยากรในการเรียนการสอน และสภาพแวดล้อมและความปลอดภัย และวิเคราะห์ความรู้สึก แสดงเป็นแดชบอร์ด

ประสบการณ์การทำงาน

  • พนักงานประกอบตู้เฟอร์นิเจอร์ (ช่วงปิดภาคการศึกษา)
    • ประกอบตู้ เตียง ซ่อมแซม และตัดแต่งอะไหล่ตู้เฟอร์นิเจอร์
  • พนักงานฝ่ายขายร้านอุปกณ์ IT (Part-Time)
    • ต้อนรับลูกค้า ให้คำแนะนำ และชำระสินค้าพร้อมส่งลูกค้า รวมถึงเคยฝึกงานในตำแหน่งเคลมสินค้า คอยตรวจเช็คสภาพสินค้า และส่งเคลมสินค้า, ตำแหน่งสต็อกสินค้า ตรวจนับสินค้าเข้า และจัดเรียง ชำระสินค้าและตัดยอดบิลสินค้าที่สั่ง, ตำแหน่งช่าง ประกอบคอมพิวเตอร์ ลงโปรแกรมต่าง ๆ
  • แคชเชียร์ (Part-Time)
    • แคชเชียร์ ณ ห้างสรรพาสินค้า คอยชำระเงิน และให้บริการโปรชั่นต่าง ๆ
  • พัฒนาแอปพลิเคชันบนมือถือ (Part-Time)
    • ออกแบบ UX/UI และพัฒนาแอปพลิเคชันบนมือถือด้วย Flutter + Dart
  • พนักงานติดตั้งกล้องวงจรปิด (งานเสริมในวันหยุด)
    • ติดตั้งกล้องวงจรปิด เดินสายแลน และติดตั้งจานดาวเทียม
